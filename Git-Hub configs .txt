To configure Git and GitHub, you can follow these steps:

1. Install Git
Download and install Git from the official website.
2. Set Up Git
Open your terminal or command prompt and configure your Git username and email:
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

3. Generate SSH Key (Optional)
If you prefer to use SSH for authentication, generate an SSH key:
ssh-keygen -t rsa -b 4096 -C "your.email@example.com"

Add the SSH key to your GitHub account by copying the key and pasting it into your GitHub settings under “SSH and GPG keys”.
4. Configure GitHub CLI (Optional)
Install GitHub CLI for additional GitHub-specific commands:
gh auth login

5. Clone a Repository
Clone a repository using HTTPS or SSH:
git clone https://github.com/username/repository.git
# or
git clone git@github.com:username/repository.git

6. Basic Git Commands
Add files to staging area:
git add .

Commit changes:
git commit -m "Your commit message"

Push changes to GitHub:
git push origin main

7. Configure Git Settings
You can customize Git settings using the git config command. For example, to set your default text editor:
git config --global core.editor "code --wait"

For more detailed information, you can refer to the Git documentation1 and GitHub Docs2.
